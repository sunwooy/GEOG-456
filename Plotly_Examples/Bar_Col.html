A bar chart is great at seeing which year is greater than another. However, it has a lot of limitation. First, because the y-axis has to start at 0 for bar charts the patterns in the upper y-axis aren't as pronounced as using a scattergram. Second, because the bars can't overlap (i.e. like a scattergram) when we have A LOT of dataset the bars get so small that they're not comprehendable. This is why I was forced to calculate the annual average elevation instead of just plotting individual points. This might hide outliers.
